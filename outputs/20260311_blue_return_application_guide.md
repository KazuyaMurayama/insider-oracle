# 所得税の青色申告承認申請書 — 提出ガイド

> **期限: 2026年3月15日（土）** → 土日のため **3月16日（月）が実質期限**
> 提出すると **2026年分の確定申告（2027年3月）** から青色申告が使えます

---

## やることは3ステップだけ

```
① 申請書を入手する（2分）
② 記入する（5分）
③ 提出する（郵送 or e-Tax or 持参）
```

---

## ① 申請書を入手する

### 方法A: PDFをダウンロードして印刷

国税庁の公式ページから PDF を入手できます：

**国税庁「所得税の青色申告承認申請書」のページ**
https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/shinkoku/annai/09.htm

→ ページ内の「申請書様式」のPDFリンクからダウンロード → 印刷

### 方法B: e-Tax（オンライン）で作成・提出

e-Taxなら印刷・郵送不要でそのまま提出できます：

**e-Tax ソフト（WEB版）**
https://www.e-tax.nta.go.jp/

→ ログイン → 「申告・申請・納税」→「操作に進む」→「申請・届出」→「所得税の青色申告承認申請」

※ e-Tax を使うにはマイナンバーカード＋ICカードリーダー（またはスマホ読取）が必要です

### 方法C: 税務署で用紙をもらう

管轄の税務署に行けば、その場で用紙がもらえます。

**江戸川税務署**
- 住所: 東京都江戸川区平井1-16-11
- 電話: 03-3683-4281
- 受付時間: 平日 8:30〜17:00

**管轄の税務署を調べる（国税庁）**
https://www.nta.go.jp/about/organization/access/map.htm

## ② 記入する

申請書は A4 で1枚です。記入する箇所と、あなたの場合の記入内容を整理します。

### 記入例（葛西物件の場合）

| 欄 | 記入内容 | 補足 |
|----|---------|------|
| **提出先** | 江戸川税務署長 殿 | 住所地の管轄税務署 |
| **提出日** | 令和8年3月○日 | 提出する日付 |
| **納税地** | 自宅の住所 | 物件の住所ではない |
| **氏名・フリガナ** | あなたの氏名 | |
| **生年月日** | あなたの生年月日 | |
| **職業** | 会社員・不動産賃貸業 | 両方書いてOK |
| **屋号** | 空欄でOK | 個人なら不要 |
| **所得の種類** | **不動産所得** にチェック ✅ | 事業所得ではない |
| **いままでに青色申告の取消しを受けたこと** | **無** | 初めてなので |
| **本年1月16日以後に業務を開始した場合** | 空欄 | 2025年に開始済み |
| **相続による事業承継の有無** | **無** | |

### 帳簿の選択（重要）

| 欄 | 選択 | 理由 |
|----|------|------|
| **簿記方式** | **✅ 複式簿記** | 会計ソフトを使えば自動で複式簿記になる |
| **備付帳簿** | **✅ 総勘定元帳** **✅ 仕訳帳** | この2つにチェック。他は任意 |

> **Agent-D**: 「複式簿記って難しそう…」と思うかもしれませんが、freee や マネーフォワードを使えば、レシートを入力するだけで勝手に複式簿記になります。手書きで帳簿をつける必要は一切ありません。

### 記入例のイメージ（国税庁の記載例）

**国税庁 記載例PDF**
https://www.nta.go.jp/taxes/tetsuzuki/shinsei/annai/shinkoku/annai/09.htm
→ ページ内の「記載要領」で書き方の見本が見られます

## ③ 提出する

3つの方法があります。どれでもOK。

### 方法A: 郵送（おすすめ — 確実＆楽）

```
送り先:
  〒132-8668
  東京都江戸川区平井1-16-11
  江戸川税務署 宛

同封するもの:
  ✅ 青色申告承認申請書（記入済み）
  ✅ 控え用のコピー1部（収受印を押して返送してもらう）
  ✅ 返信用封筒（切手を貼って自分の住所を書く）

※ 3/16必着なので、3/13（木）までに投函すると安全
```

> **ポイント**: 控えを返送してもらうために、返信用封筒（84円切手貼付）を同封してください。控えは「ちゃんと届いた証拠」になります。

### 方法B: e-Tax（最速 — 期限ギリギリでもOK）

e-Tax なら **3/16の23:59まで** 提出可能です。

1. e-Tax にログイン
2. 「申請・届出」→「所得税の青色申告承認申請」
3. 画面の指示に従って入力
4. 送信 → 完了

**e-Tax ログイン**
https://www.e-tax.nta.go.jp/

### 方法C: 税務署に直接持参

江戸川税務署の窓口に持っていく方法です。

- 平日 8:30〜17:00
- 時間外は「時間外収受箱」に投函可能（税務署の入口付近にあるポスト）

## よくある疑問

### Q: 届出を出したら、必ず青色申告しないとダメ？

**いいえ。** 届出を出しても、確定申告の時に白色で出すことも可能です。届出はあくまで「青色で申告する権利を得る」だけです。デメリットはありません。

### Q: 会計ソフトは何がいい？

| ソフト | 月額 | 特徴 |
|--------|------|------|
| freee | 約1,000円/月 | スマホ対応◎。初心者に最もやさしい |
| マネーフォワード クラウド確定申告 | 約1,000円/月 | 銀行連携が強い |
| やよいの青色申告オンライン | 初年度無料 | コスパ◎。不動産所得に対応 |

> いずれも年間約12,000円で、**その費用自体が経費**になります。

### Q: 届出を出し忘れたらどうなる？

2026年分の青色申告は使えません。次に出せるのは2027年3月15日で、2027年分からの適用になります。**1年分の節税チャンスを失います。**
