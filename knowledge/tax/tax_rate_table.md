## 法人化判断テーブル（Agent-B監修）

### 法人化のメリット・デメリット

| 項目 | 個人 | 法人 |
|------|------|------|
| 最高税率 | 所得税45%+住民税10%=55% | 法人税実効約33% |
| 損失繰越 | 3年（青色申告） | 10年 |
| 経費の幅 | 限定的 | 広い（役員報酬・退職金等） |
| 社会保険 | 給与の社保のみ | 法人で加入（コスト増） |
| 設立費用 | 不要 | 約25〜30万円 |
| 維持費用 | 確定申告のみ | 法人税申告（税理士費年30〜50万） |
| 融資 | 個人信用 | 法人+個人保証 |
| 相続 | 不動産そのもの | 株式の評価額 |

### 🔴 法人化の損益分岐判断フロー

```
Q1: 不動産所得（給与と合算後の課税所得）が900万円を超えるか？
  → NO → 個人のまま（法人化メリットなし）
  → YES → Q2へ

Q2: 今後5年以内に物件を追加取得する予定か？
  → NO → 個人のまま（法人維持コストが割に合わない）
  → YES → Q3へ

Q3: 年間の不動産所得（経費・償却前の利益）が500万円を超えるか？
  → NO → まだ早い。物件追加後に再検討
  → YES → 法人化を具体的に検討すべき
```

### 年収×不動産規模 マトリクス（法人化判断）

| | 物件1〜2棟 | 物件3〜5棟 | 物件6棟以上 |
|---|-----------|-----------|-----------|
| **年収〜700万** | 個人✅ | 個人✅ | 法人検討🔶 |
| **年収700〜1000万** | 個人✅ | 法人検討🔶 | 法人推奨🔴 |
| **年収1000〜1500万** | 個人✅ | 法人推奨🔴 | 法人必須🔴 |
| **年収1500万超** | 法人検討🔶 | 法人推奨🔴 | 法人必須🔴 |

### Agent-C警告: 法人化と融資の関係

- **法人化直後は融資が通りにくくなる**（法人の実績がゼロ）
- 対策: 法人化前に次の物件の融資を個人で通しておく
- 法人設立1〜2期目は決算書がないため、個人の保証力が重要
- **法人化のタイミングは「物件購入の直後」ではなく「次の購入の前」が理想**

### マイクロ法人の年間維持コスト概算
```
法人住民税（均等割）: 約7万円/年（赤字でも必ず発生）
税理士費用: 30〜50万円/年
社会保険料（役員分）: 役員報酬の約30%
登記変更等: 数万円/年
合計: 最低でも年間50〜80万円の固定コスト
```
