# デッドクロス計算モデル

## デッドクロスとは

ローン返済の元本部分が減価償却費を上回る時点のこと。
この時点を境に、**キャッシュアウトしているのに帳簿上は黒字（＝税金が発生）**という逆転現象が起きる。

```
【正常期】 減価償却費 > 元本返済 → 帳簿赤字（節税効果あり）
【デッドクロス後】 元本返済 > 減価償却費 → 帳簿黒字（税金発生）
```

## 🔴 なぜ危険か（Agent-B解説）

- ローン元本返済は「経費にならない支出」
- 減価償却費は「支出のない経費」
- デッドクロス後は「手元にお金が残らないのに税金を払う」状態になる
- 最悪のケースでは「税金を払うために借金する」ことになる

## 計算モデル

### 前提条件の設定
```
P: 物件価格
B: 建物価格 (= P × 建物比率)
L: 借入額
r: 年利
n: 返済年数
D: 耐用年数（減価償却期間）
```

### 年間減価償却費
```
年間償却費 = B ÷ D

例: 建物1,200万 ÷ 4年 = 300万/年（築古木造の場合）
```

### 元利均等返済における年間元本返済額
```
年間返済額（元利合計） = L × r(1+r)^n ÷ ((1+r)^n - 1)

元本部分は毎年増加し、利息部分は毎年減少する。
初年度の元本比率 < 最終年度の元本比率
```

### デッドクロス発生時期の計算

元利均等返済の場合、k年目の年間元本返済額は近似的に：
```
元本返済(k年目) ≈ 年間返済額 - L × r × (1 - (k-1)/n)
```

デッドクロス発生年 = 「元本返済額 ≧ 減価償却費」となる最初の年

## パターン別デッドクロスシミュレーション

### パターン1: 築古木造（最も危険）
```
物件: 2,000万（建物1,200万・土地800万）
構造: 木造・築25年 → 耐用年数4年
借入: 1,800万・金利2%・20年返済

年間償却費: 300万/年（1〜4年目のみ）
年間返済額: 約110万/年

→ デッドクロス: 5年目に突然発生（償却が終了するため）
→ 1〜4年目: 償却300万 >> 元本90万 → 大幅節税
→ 5年目〜: 償却0万 << 元本95万 → 一気に増税
```

⚠️ **Agent-A警告**: 築古木造4年償却は「4年で節税メリットが消える時限爆弾」。5年目以降の出口戦略なしに買うのは極めて危険。

### パターン2: 築15年RC（安定型）
```
物件: 5,000万（建物3,000万・土地2,000万）
構造: RC・築15年 → 耐用年数: (47-15)+15×0.2 = 35年
借入: 4,500万・金利1.8%・30年返済

年間償却費: 約86万/年（35年間継続）
年間返済額: 約194万/年

→ 初年度の元本返済: 約113万 → 既に償却費を超過
→ 実はRC中古はデッドクロスが「最初から」発生していることが多い
→ ただし家賃収入でカバーできるため、問題にならないケースも多い
```

### パターン3: 新築木造（中間型）
```
物件: 3,000万（建物1,800万・土地1,200万）
構造: 木造・新築 → 耐用年数22年
借入: 2,700万・金利2%・25年返済

年間償却費: 約82万/年（22年間）
年間返済額: 約137万/年

→ デッドクロス: 約12〜15年目
→ 比較的緩やかに発生するため、対策を立てやすい
```

## デッドクロス対策（4エージェント統合見解）

### Agent-B: 税務設計での対策
1. **繰上返済**: デッドクロス前に元本を減らす
2. **物件追加**: 新しい物件の償却費でカバー
3. **設備分離**: 建物附属設備を分離し、15年償却で平準化
4. **法人化**: 法人なら損失繰越が10年（個人は3年）

### Agent-C: 融資面での対策
1. **デッドクロス前の物件追加購入**: 融資枠があるうちに次を買う
2. **借換え**: 返済期間の延長で元本返済を平準化
3. **注意**: 繰上返済すると手元資金が減り、次の融資に不利

### Agent-D: 実務面での対策
1. **5年目の売却**: 短期→長期譲渡の切替タイミングで売却
2. **家賃値上げ**: 設備投資による付加価値向上
3. **現実的には**: 築古木造は「4年で売る前提」で買うのが正解だった（経験談）

### Agent-A: 税務調査面での注意
1. **意図的な赤字計上の継続は危険**: 3年以上赤字が続くと「事業性」を疑われる
2. **売却時の注意**: 減価償却した分は売却時に「取得費から差し引かれる」ため、譲渡所得が増える
3. **5年超の長期譲渡にすることが重要**: 税率が39%→20%に下がる
