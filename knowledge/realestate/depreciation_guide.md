# 減価償却設計ガイド

不動産投資における減価償却の最適設計を物件タイプ別に整理する。

## 法定耐用年数一覧

| 構造 | 用途 | 耐用年数 | 定額法償却率 |
|------|------|----------|-------------|

| 重量鉄骨造 | 住宅用 | 34年 | 0.030 |
| 軽量鉄骨造 | 住宅用 | 19年 | 0.053 |
| 木造 | 住宅用 | 22年 | 0.046 |

## 中古物件の耐用年数計算

### 法定耐用年数を超過した場合
```
耐用年数 = 法定耐用年数 × 0.2
例: 築25年木造 → 22 × 0.2 = 4年（償却率 0.25）
```

### 法定耐用年数内の場合
```
耐用年数 = (法定耐用年数 - 経過年数) + 経過年数 × 0.2
例: 築10年木造 → (22-10) + 10×0.2 = 14年
```

## 建物・設備の按分（Agent-B推奨）

物件取得価格を建物と土地に按分する際のポイント：
- 固定資産税評価額の比率が基本
- 売買契約書に建物価格が明記されている場合はそれを優先
- 設備（給排水・電気・エレベーター等）を建物本体と分離すると償却を加速できる

## 🔴 Agent-A警告

- 建物比率を不自然に高くする按分は否認リスクあり
- 設備分離は合理的な根拠（見積書・鑑定書）が必要
- 築古木造4年償却は税務署が注目するパターンの一つ
