## 3. 建物本体の減価償却

### 3-1. 耐用年数の計算

RC造住宅の法定耐用年数: **47年**

中古資産の耐用年数（簡便法）:
```
経過年数 = 30年（1995年築 → 2025年取得）
法定耐用年数 47年 > 経過年数 30年 なので：

残存耐用年数 = 47年 − 30年 + 30年 × 20%
             = 17年 + 6年
             = 23年
```

> ※ 端数処理: 1年未満の端数は切り捨て
> ※ 根拠法令: 耐用年数省令 第3条第1項

**採用耐用年数: 23年**（定額法償却率: 0.044）

> **注**: properties/kasai_rc_603.json には 27年と記載されているが、
> 取得時の正確な築年数に基づき税理士と再計算の上、確定値を採用すること。

### 3-2. 建物本体の年間償却額（設備分離なしの場合）

```
建物取得価額: 11,866,000円
耐用年数: 23年
償却率: 0.044

年間償却額 = 11,866,000 × 0.044 = 522,104円
```

## 4. 附属設備分離（重要）

### 4-1. なぜ設備分離が必要か

建物本体の償却だけでは、土地利子制限（租税特別措置法第41条の4）により
不動産所得の赤字が給与所得と損益通算できない可能性が高い。

設備を分離して短い耐用年数で償却することで、
償却費を増加させ、土地利子制限を超える赤字を確保する。

### 4-2. 設備分離の根拠

#### 方法: 都税事務所の再建築費評点数内訳を利用

都税事務所で閲覧できる「再建築費評点数」の内訳から、
建物本体と附属設備の比率を算出する。

| 区分 | 再建築費評点数 | 比率 |
|------|--------------|------|
| 建物本体（躯体・仕上） | ※都税事務所で確認 | ※% |
| 附属設備（電気・給排水・空調等） | ※都税事務所で確認 | ※% |
| **合計** | — | **100%** |

> **重要**: 都税事務所で閲覧した評点数の値を必ずここに転記すること。
> 閲覧日・担当者名もメモしておくと、税務調査時の説得力が増す。

| 閲覧場所 | 江戸川都税事務所 |

### 4-3. 設備部分の減価償却

#### 附属設備の耐用年数

RC造の附属設備（電気設備・給排水設備・空調設備等）:
- 法定耐用年数: **15年**

中古資産の耐用年数（簡便法）:
```
経過年数 = 30年
法定耐用年数 15年 < 経過年数 30年 なので：

残存耐用年数 = 15年 × 20% = 3年
```

**採用耐用年数: 3年**（定額法償却率: 0.334）

#### 設備部分の取得価額（仮計算 — 設備比率20%の場合）

```
建物取得価額: 11,866,000円
設備比率: 20%（※都税事務所の評点数比率で確定）

設備取得価額 = 11,866,000 × 20% = 2,373,200円
建物本体取得価額 = 11,866,000 × 80% = 9,492,800円
```

| 区分 | 取得価額 | 耐用年数 | 償却率 | 年間償却額 |
|------|---------|---------|--------|-----------|

## 5. 土地利子制限の影響

### 計算（設備分離ありの場合）

```
家賃収入:        1,860,000円（155,000 × 12ヶ月）
経費（償却除く）: 約1,300,000円
減価償却費:      1,210,332円（設備分離あり）

不動産所得 = 1,860,000 - 1,300,000 - 1,210,332 = △650,332円（赤字）

土地利子 = 借入金 × 土地比率 × 金利
         = 33,500,000 × 65.1% × 2.25% ÷ 12 × 12
         = 約490,000円

損益通算可能額 = 赤字 650,332 - 土地利子 490,000 = 160,332円

→ 設備分離により約160,000円の損益通算が可能に
→ 年収800万の場合: 160,000 × 約33% = 約53,000円の節税
```

### 計算（設備分離なしの場合）

```
不動産所得 = 1,860,000 - 1,300,000 - 522,104 = 37,896円（黒字）

→ 損益通算不可。節税額ゼロ。
```
