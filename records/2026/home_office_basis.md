# 自宅按分の根拠書類（2026年）

> この書類は、自宅関連経費の按分率の根拠を税務署に説明するための記録です。
> 税務調査で「なぜこの按分率なのか」と聞かれた際に、この書類を提示します。

---

## 1. 自宅の基本情報

| 項目 | 内容 |
|------|------|
| 所在地 | （自宅住所） |
| 間取り | （例: 2LDK） |
| 総面積 | （例: 55㎡） |
| 居住者 | 本人＋配偶者 |
| 賃貸 or 持家 | 賃貸（月額120,000円） |

## 2. 事業使用スペース

| 項目 | 内容 |
|------|------|
| 使用場所 | リビング内のデスクスペース |
| 面積 | 約5.5㎡（デスク＋書棚＋プリンター設置場所） |
| 按分率（面積比） | 5.5㎡ ÷ 55㎡ = **10%** |

### 事業使用の内容
- 不動産投資に関する帳簿記入・会計ソフト入力
- 管理会社・税理士とのメール・電話連絡
- 物件情報の調査・分析
- 確定申告書類の作成
- 領収書・契約書の保管・整理

### 【添付すべき資料】
- [ ] 自宅の間取り図（事業使用エリアをマーカーで囲む）
- [ ] デスクスペースの写真（日付入り）

## 3. 各経費の按分率と計算

### 家賃
| 項目 | 金額 |
|------|------|
| 月額家賃 | 120,000円 |
| 按分率 | 10%（面積比） |
| 月額経費 | 12,000円 |
| **年間経費** | **144,000円** |

### 光熱費（電気・ガス・水道）
| 項目 | 金額 |
|------|------|
| 月額光熱費（平均） | 30,000円 |
| 按分率 | 10%（面積比に準拠） |
| 月額経費 | 3,000円 |
| **年間経費** | **36,000円** |

> 按分率の根拠: 家賃と同じ面積比を適用。
> 光熱費を面積比と異なる率にすると「根拠が不統一」と指摘されるリスクがあるため、
> 同一の按分率を使用する。

### インターネット + スマートフォン
| 項目 | 金額 |
|------|------|
| 月額通信費 | 10,000円（ネット6,000円 + スマホ4,000円） |
| 按分率 | 20% |
| 月額経費 | 2,000円 |
| **年間経費** | **24,000円** |

> 按分率の根拠: 家賃・光熱費（10%）より高い理由は以下の通り。
> - インターネット: 物件検索（SUUMO・楽待等）、管理会社とのメール、
>   ネットバンキング（家賃入金確認・振込）、会計ソフトの操作、
>   不動産投資の情報収集に直接使用
> - スマートフォン: 管理会社との電話連絡、物件視察時の写真撮影、
>   領収書のスキャン保存、交通費のICカード履歴確認
> - 上記の使用頻度から、通信環境全体の20%を事業使用と判断
