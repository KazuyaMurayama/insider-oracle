> **このフォルダの記録が、確定申告の経費を守る「証拠」になります。**
> 記録がなければ、どんなに正当な経費でも否認されます。

## 記録のルール

1. **日付・金額・目的**の3点は必ず記録する
2. レシート・領収書はスマホで撮影し、クラウド保存する
3. 交通系ICカードの履歴も証拠になる（印字期限があるので四半期ごとに印字）
4. CSVファイルは月末に振り返りで抜け漏れを補完する

## 固定費（記録不要 — 自動引落・契約書で証明）

以下は毎月自動で発生するため、このフォルダでの記録は不要です。
納税通知書・契約書・ローン返済予定表を保管しておけばOK。

- 管理費・修繕積立金（月30,000円）
- 賃貸管理委託費（月2,500円）
- ローン返済（月115,000円・うち利息は返済予定表で確認）
- 固定資産税・都市計画税（年234,845円）
- 火災保険（年20,000円）
- 不動産取得税（届いた額をそのまま）
