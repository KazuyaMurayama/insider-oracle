# 特化モジュール：サラリーマン不動産投資節税

> 不動産投資・節税の質問が来た時に参照。
> このケースでは `agents/` 配下の4名固定エージェントを使用する。

## 固定エージェント（不動産節税専用）

- `agents/agent_a_tax_inspector.md` — 元国税調査官
- `agents/agent_b_tax_accountant.md` — 富裕層専門税理士
- `agents/agent_c_bank_officer.md` — 地方銀行融資担当
- `agents/agent_d_landlord.md` — 元サラリーマン大家

## 質問の分類カテゴリ

```
A: 経費計上（何が経費になるか）
B: 減価償却設計（物件・設備の償却戦略）
C: 損益通算（給与との合算節税）
D: 法人化判断（個人→法人の最適タイミング）
E: 税務調査対策（リスク管理・証拠整備）
F: 出口戦略（売却時の税金・デッドクロス対策）
G: その他
```

## 追加コマンド（不動産投資特化）

| コマンド | 内容 |
|----------|------|
| `/simulate [年収] [物件規模]` | 節税シミュレーション実行（詳細は `.claude/rules/simulate-framework.md`） |
| `/check [経費項目]` | その経費が認められるか判定 |
| `/risk [戦略名]` | 税務調査リスク評価 |
